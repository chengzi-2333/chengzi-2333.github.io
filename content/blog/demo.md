## Maxwell's equation

Differential form:
$$
\nabla \cdot \textbf{E} = \frac{ \rho_{e} }{ \epsilon_{0} }
$$
$$
\nabla \cdot \textbf{B} = 0
$$
$$
\nabla \times \textbf{E} = - \frac{ \partial{ \textbf{B} } }{ \partial{t} }
$$
$$
\nabla \times \textbf{B} = \mu_{0} \textbf{J} + \mu_{0} \epsilon_{0} \frac{ \partial{ \textbf{E} } }{ \partial{t} }
$$

Integral form:
$$
\oint_{S}{ \textbf{E} \cdot \mathrm{d} \textbf{S} } = \frac{Q}{ \epsilon_{0} }
$$
$$
\oint_{S}{ \textbf{B} \cdot \mathrm{d} \textbf{S}} = 0
$$
$$
\oint_{C}{ \textbf{E} \cdot \mathrm{d} \textbf{l} } = - \int_{S}{ \frac{ \partial{ \textbf{B} } }{ \partial{t} } \cdot \mathrm{d} \textbf{S} }
$$
$$
\oint_{C}{ \textbf{B} \cdot \mathrm{d} \textbf{l} } = \mu_{0} \mathrm{I} + \mu_{0} \epsilon_{0} \int_{S}{ \frac{ \partial{ \textbf{E} } }{ \partial{t} } \cdot \mathrm{d} \textbf{S} }
$$
